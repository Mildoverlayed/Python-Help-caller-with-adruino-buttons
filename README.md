1. Project Title
Python Help Caller
2. Overview
- This was a project assigned for me to make and my goal was to create a way for buttons to create an help caller when pressed
3. Features
- automatic connection to mutiple microcontrollers at a time over usb serial
- theritical infinite amount of microcontorllers and buttons connected
- dynamic updating and connection
- a all in one suite that creates it own window for help calling
- a cloud based way for Google Sheets with configuaation and storage in said sheet
- basic backend as a way to see how the project works in more of a redmentary way

- Run included Python script
